# zsnoop-mcp

Operate the upstream `zsnoop-mcp` server for ZFS snapshot discovery, snapshot
diffing, file reads, fetch-based recovery, and opt-in live restores.

## Files

- `SKILL.md` - trigger and workflow guidance.
- `agents/openai.yaml` - OpenAI/Codex UI metadata.
- `references/configuration.md` - host config, SSH, sudo, and restore setup.
- `references/tools.md` - tool selection, limits, and safety notes.
